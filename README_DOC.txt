### Building the pdf documentation 

In order to build the documentation in pdf format, you will need:

- A latex distribution installation

The following extra  packages are required:

- titlesec
- framed
- threeparttable
- wrapfig
- capt-of
- needspace
- multirow
- eqparbox
- environ
- trimspaces
- URW 'base35' font pack for Latex
